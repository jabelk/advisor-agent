## Lesson Curriculum

### Lesson Progression

| # | Topic | Salesforce Concept | MCP Verification Tool | Builds On |
|---|-------|-------------------|----------------------|-----------|
| 0 | Getting Started | Sandbox login, navigation | `sandbox_seed_clients` | — |
| 1 | Contacts | View, browse, understand contact records | `sandbox_get_client`, `sandbox_list_clients` | Lesson 0 |
| 2 | Tasks | Create follow-up tasks, track activities | `sandbox_create_task`, `sandbox_show_tasks` | Lesson 1 |
| 3 | Search & Filter | Find clients by criteria, use search bar | `sandbox_search_clients`, `sandbox_query_clients` | Lesson 1 |
| 4 | List Views | Create filtered views, save for reuse | `sandbox_save_listview`, `sandbox_show_listviews` | Lesson 3 |
| 5 | Reports | Build reports with filters, view in Salesforce | `sandbox_save_report`, `sandbox_show_reports` | Lesson 4 |

### Lesson Template Structure

Each lesson follows this consistent structure:

1. **Objective** — What you'll learn and why it matters for any CRM
2. **CRM Concept** — The transferable idea (not Salesforce-specific)
3. **Prerequisites** — Which earlier lessons you need
4. **Exercise: Do It Manually** — Step-by-step in Salesforce sandbox
5. **Verify with Claude** — Ask Claude Desktop to check your work via MCP
6. **Key Takeaways** — What transfers to any CRM platform
7. **Challenge** — Unguided exercise to try independently

### CLAUDE.md Tutor Prompt Design

The CLAUDE.md instructs Claude to:
- Act as a Salesforce tutor for a financial advisor learning CRM basics
- Know the lesson curriculum (topics, order, MCP tools per lesson)
- Proactively use MCP tools when Jordan asks to verify work
- Explain CRM concepts in terms relevant to financial advisors
- Reference specific seed data clients by name in examples
- Guide Jordan through exercises without giving away answers immediately
- Suggest the next lesson when one is completed
